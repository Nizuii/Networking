# What is IP Address?

- IP Address is a layer 3 logical address assigned by an administrator.
- IP Address is used to uniquely identify devices on a network.
- Every device on the internet has a unique IP address.
- Ip address has 2 versions:

  1. **IPV4** - Internet Protocol Version 4
  1. **IPV6** - Internet Protocol Version 6
 
- IP is a connectionless protocol unlike TCP which is a connection oriented protocol.
